**Online Learning Management System & Course Marketplace**

Create, sell, and deliver online courses with video hosting, quizzes, certificates, and analytics.

## Revenue Model
- Platform commission (5-30% per course sale)
- Instructor pro plans ($49/month)
- Enterprise team subscriptions
- Featured course placements

## Features
- Course creation with drag-and-drop curriculum builder
- Video lesson hosting with progress tracking
- Interactive quizzes with auto-grading
- Multiple question types (MCQ, True/False, Short Answer)
- Auto-generated completion certificates
- Student progress & analytics dashboard
- Course discussion forums
- Student notes with video timestamps
- Course reviews & ratings
- Coupon & discount system
- Category-based course browsing
- Full-text search
- Instructor revenue dashboard
- Stripe payment processing

## Getting Started
```bash
cd 08-LearnPath && npm install && npx prisma migrate dev && npm run dev
```
