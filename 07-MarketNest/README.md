**Multi-Vendor E-Commerce Marketplace Platform**

Launch your own marketplace. Sellers list products, buyers shop, you earn commission on every sale.

## Revenue Model
- Transaction fees (5-15% per sale)
- Seller subscriptions ($29-$199/month)
- Promoted listings & advertising

## Features
- Multi-vendor onboarding & store management
- Product catalog with images, variants, & categories
- Shopping cart with multi-vendor checkout
- Stripe Connect for split payments & payouts
- Order management per vendor
- Product reviews & seller ratings
- Coupon & discount system
- Wishlist functionality
- Address management
- Order tracking & notifications
- Admin dashboard with analytics
- SEO-optimized product pages

## Getting Started

cd 07-MarketNest && npm install && npx prisma migrate dev && npm run dev
