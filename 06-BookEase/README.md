**Appointment Booking & Scheduling Platform**

Let clients book appointments 24/7. Built for salons, clinics, consultants, fitness studios, and service businesses.

## Revenue: Free / $29/mo / $79/mo

## Features
- Beautiful branded booking page
- Service & staff management
- Availability-based scheduling with conflict detection
- Online payment collection via Stripe
- Email & SMS appointment reminders
- Customer management & history
- Group booking support
- Coupon & discount codes
- Customer reviews & ratings
- Calendar sync (iCal export)
- Multi-location support
- Cancellation & rescheduling policies
- Business analytics dashboard

cd 06-BookEase && npm install && npx prisma migrate dev && npm run dev
