**Enterprise Inventory & Warehouse Management System**

Real-time stock tracking across multiple warehouses with barcode scanning, purchase orders, and smart alerts.

## Revenue Model
- **Starter**: $29/month (1 warehouse, 500 products)
- **Business**: $79/month (5 warehouses, unlimited products, POs)
- **Enterprise**: $199/month (unlimited, batch tracking, integrations)

## Features
- Multi-warehouse stock management
- Zone & location tracking (aisle/shelf/bin)
- Barcode & QR code scanning
- Product variants (size, color, etc.)
- Real-time stock levels & availability
- Stock movements with audit trail
- Low-stock & expiry alerts
- Purchase order management
- Supplier management
- Stock transfers between warehouses
- Cycle counting & stock takes
- Inventory valuation reports
- Excel import/export
- Batch & lot tracking
- Perishable goods management

## Getting Started
```bash
cd 04-StockSync
npm install
npx prisma migrate dev
npm run dev
```
