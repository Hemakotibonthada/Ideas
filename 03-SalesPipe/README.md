**Enterprise CRM & Sales Pipeline Management**

Track leads, manage deals, and close more sales with visual pipeline management and powerful analytics.

## Revenue Model
- **Starter**: $15/user/month (500 contacts, 2 pipelines)
- **Growth**: $39/user/month (10K contacts, automation, lead scoring)
- **Enterprise**: $79/user/month (unlimited, SSO, forecasting)

## Features
- Visual drag-and-drop sales pipeline
- Contact & company management with full profiles
- Lead scoring & qualification
- Deal tracking with probability forecasting
- Activity logging (calls, emails, meetings)
- Email tracking with open/click notifications
- Sales task management
- Custom fields for contacts & deals
- Tag-based organization
- Revenue forecasting & analytics
- Team performance dashboards
- Workflow automation
- API access for integrations

## Tech Stack
- Next.js 14, React, TypeScript, Tailwind CSS
- Prisma ORM + PostgreSQL
- Chart.js for analytics
- NextAuth.js

## Getting Started
```bash
cd 03-SalesPipe
npm install
cp .env.example .env
npx prisma migrate dev
npm run dev
```
